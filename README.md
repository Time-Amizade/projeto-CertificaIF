RESUMO

No Instituto Federal do Paraná, as atividades complementares, que envolvem a participação dos estudantes em eventos, cursos e projetos, são obrigatórias para a conclusão do curso e, atualmente, seu controle é realizado de forma manual. O projeto originou-se a partir da experiência dos próprios alunos que enfrentam dificuldades e atrasos ao enviar comprovantes de forma manual através do site do IFPR, o que gera sobrecarga administrativa. O projeto CertificaIF, desenvolvido no Instituto Federal do Paraná – Campus Foz do Iguaçu, teve como objetivo criar um sistema web para gerenciar as atividades complementares exigidas nos cursos da instituição, propondo centralizar e informatizar o processo e permitindo que os estudantes enviem seus comprovantes digitalmente, acompanhem a validação por parte da instituição e visualizem a carga horária já cumprida. Para o desenvolvimento do sistema, foram utilizadas ferramentas como PHP, MySQL, Figma, Jira, GitHub, HTML, CSS, JavaScript e LucidChart (UML). O resultado consistiu na implementação de um sistema web capaz de centralizar o envio digital dos comprovantes, apoiar a validação institucional das atividades complementares e disponibilizar aos estudantes o acompanhamento da carga horária integralizada.
